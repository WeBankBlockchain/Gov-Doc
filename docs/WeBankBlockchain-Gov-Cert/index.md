# 证书管理组件
[![License](https://img.shields.io/badge/license-Apache%202-4EB1BA.svg)](https://www.apache.org/licenses/LICENSE-2.0.html)

```eval_rst
.. admonition:: **简介**

    WeBankBlockchain-Gov-Cert提供了证书生命周期管理的解决方案，规范证书签发流程，支持证书托管，支持多种签名算法，方便个人或企业使用。

```

```eval_rst
.. admonition:: **主要特性**

    - 支持多种密钥和签名算法
    - 支持证书托管
    - 支持多级证书签发
    - 支持证书重置
    
```
